# Windows Install

This page is a draft, high-level guide for the Windows build.

## Basic Flow

1. Download the latest BUS Core release from the project's release page.
2. Extract the release contents to a normal local folder.
3. Run BUS Core using the included Windows release entrypoint.
4. Let BUS Core open its local browser UI.

## What To Expect

- BUS Core is local-first.
- Your working data is intended to stay on your local machine unless you deliberately move or back it up.
- The browser UI should open locally rather than sending you to a hosted account portal.
