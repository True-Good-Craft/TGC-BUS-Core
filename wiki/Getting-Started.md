# Getting Started

This page is a simple starting point for BUS Core beta users.

## Choose Your Install Path

- [Windows Install](Windows-Install.md) for a local Windows setup.
- [Docker Install](Docker-Install.md) for a container-based setup.
- [Synology NAS Docker Setup (Beta)](Synology-NAS-Docker-Setup-Beta.md) if you are specifically testing on Synology NAS.

## First Goals

When BUS Core opens for the first time, a useful first pass is:

1. Confirm that the UI loads in your browser.
2. Confirm where your local data will live.
3. Try one real workflow you care about, such as inventory, jobs, costing, or general shop operations tracking.
4. Note where you felt confident, confused, or blocked.
