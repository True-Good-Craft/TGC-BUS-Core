# Contributing to BUS Core

Thank you for helping improve BUS Core.

This project welcomes contributions from developers, makers, and operators who want to build better tools for small manufacturing environments.

---

# Ways to Contribute

You can contribute by:

* reporting bugs
* improving documentation
* submitting patches
* proposing features
* improving manufacturing workflows

# Developer Certificate of Origin (DCO)

BUS Core uses the **Developer Certificate of Origin**.

[https://developercertificate.org/](https://developercertificate.org/)

By submitting a contribution you confirm that:

1. You wrote the code or documentation yourself, or
2. You have the right to submit it under the project’s license.

Each commit must include a sign-off line:

```
Signed-off-by: Your Name <your@email.com>
```

Example:

```
Signed-off-by: Alex Smith <alex@example.com>
```

---

# Code Guidelines

Please prioritize:

* reliability
* readability
* maintainability
* practical functionality

BUS Core values **simplicity over unnecessary complexity**.
