# Maintainer Notice

BUS Core is an open-source project maintained by **True Good Craft** and community contributors.

Maintainers develop and review the software but do **not operate installations for users**.

---

## Maintainers Are Not Service Providers

Maintainers do not:

* host BUS Core for users
* manage user databases
* operate installations
* provide operational monitoring

When you run BUS Core, it runs on **your own infrastructure**.

---

## No Operational Responsibility

Maintainers are not responsible for how the software is used in business operations.

This includes:

* inventory planning
* manufacturing decisions
* cost calculations
* pricing decisions
* financial reporting

BUS Core is a tool, not a managed service.

---

## No Access to User Data

BUS Core is designed as a **local-first system**.

Maintainers do not receive or store user operational data.

Because of this architecture, maintainers cannot:

* access installations
* recover databases
* repair user environments

Users are responsible for maintaining backups.
